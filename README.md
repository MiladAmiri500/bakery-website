# Ayoob Bakery Australia Website

A full-stack Node.js e-commerce site for a bakery, built with Express, MongoDB, EJS, and more. Features include product listings, cart/wishlist, user auth (local/Google), admin panel, and responsive design.

## Setup
1. Clone the repo: `git clone https://github.com/MiladAmiri500/bakery-website.git`
2. Install dependencies: `npm install`
3. Set up `.env` (see `.env.example` for templateâ€”create this file manually with your DB URI, etc.)
4. Run: `npm start`

## Tech Stack
- Backend: Express.js, Mongoose
- Frontend: EJS, Swiper.js, Font Awesome
- Auth: Passport.js
- DB: MongoDB
